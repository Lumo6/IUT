Documentation des constructeurs et destructeur

◆ CCircularBuffer()

CCircularBuffer ( size_t size = 0)

Constructeur.

Crée un buffer circulaire en spécifiant sa taille

Paramètres
sizeTaille du buffer
Remarques
La taille du buffer doit être une puissance de 2 afin d'optimiser le cycle du buffer. Dans le cas contraire, une exception CCircularBufferError de code CCircularBufferError::ErrorCodes::SizeIsNotPowerOf2 est lancée.

Documentation des fonctions membres

◆ read()

size_t read ( std::vector< unsigned char > & data)

Lit et consomme des données dans le buffer.

Cette fonction retourne les données stockées dans le buffer en les linéarisant. En interne du buffer circulaire, les données lues sont libérées de façon à pouvoir stocker de nouvelles données.

Paramètres
[in,out]dataBuffer utilisé pour stocker les données lues. En entrée, la taille de data est utilisée pour connaitre le nombre max de données à lire. En sortie, les données lues sont stocké au début du tableau data. Aucune réallocation n'a lieu lors de cet appel.
Renvoie
Le nombre de données effectivement lues. Si cette valeur est inférieure à la taille de data, alors seules les premières valeurs de data sont valides.

◆ unrollTo()

void unrollTo ( std::vector< unsigned char > & linearBuffer)

Déroule le buffer circulaire dans un buffer linéaire.

Déroule les données utiles du buffer circulaire dans un buffer linéaire. Les données ne sont pas consommées, ce qui fait que l'appel de cette fonction n'impacte pas les appels de la fonction read().

Paramètres
[in,out]linearBufferVecteur recevant les données du buffer circulaire.
Remarques
Les données du buffer circulaire sont ajoutées à la fin du vecteur linearBuffer. Des réallocations du vecteur linearBuffer sont donc possibles.
