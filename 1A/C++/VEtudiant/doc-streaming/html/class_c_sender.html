Description détaillée

Simule un serveur de streaming.

Cette classe est utilisée pour simuler l'envoie de données en streaming. Les données sont simulées par une longue chaîne de caractères découpée en paquets correspondants aux mots composant la chaîne.

Paramètres
maxSizeTaille maximal du paquet que l'on peut recevoir
Renvoie
Un vecteur de char constituant le paquet transmit
Remarques
Si le mot à envoyer est plus grand que maxSize, alors un vecteur vide est retourné et le paquet est placé en attente.
