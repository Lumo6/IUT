Description détaillée

Classe "métier" du TP.

Cette classe concentre toutes les actions du programme nécessaires pour le fonctionnement du programme. Les méthodes publiques sont les points d'entrées événementiels qui sont appelés par le programme principal en réponse des interactions de l'utilisateur.

La classe CTP est déclarée dans le fichier TP.h.
