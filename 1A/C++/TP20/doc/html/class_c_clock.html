Description détaillée

Classe représentant une horloge.

Cette classe gère une horloge au sens général. Une horloge représente le temps sur un fuseau horraire. Cette classe est destinée à être dérivée pour spécialiser l'horloge en horloge mécanique ou horloge électronique.

La classe CClock est déclarée dans le fichier Clock.h.

#include "Clock.h"

◆ AfficherInfos()

void AfficherInfos ( ) const

Affiche les informations détaillées de cette horloge.

Les informations détaillées sont affichées dans un rectangle en haut au milieu de la fenêtre. Ces informations dépendent du type d'horloge.

  • Les horloges electroniques Une horloge électronique doit afficher les informations suivantes :
    "Je suis une horloge automatique."
    "Selon moi, il est XX:XX."
  • Les horloges mécaniques Une horloge mécanique doit afficher les informations suivantes :
    "Je suis une horloge mécanique. Mon ressort est tendu à XX%"
    "Selon moi, il est XX:XX."
