Description détaillée

Classe représentant une horloge mécanique.

Cette classe gère une horloge mécanique. En plus des caractéristiques générales des horloges, une horloge mécanique est caractérisée par son ressort moteur qui se détend au fil du temps. Il faut régulièrement "remonter" l'horloge pour qu'elle continue à fonctionner.

La classe CMechClock est déclarée dans le fichier MechClock.h.

#include "MechClock.h"
