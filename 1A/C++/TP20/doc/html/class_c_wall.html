Description détaillée

Classe représentant un mur.

Cette classe gère un mur sur lequel sont suspendues des horloges. Les horloges peuvent être de différentes sortes (électroniques ou mécaniques). Un mur est affiché graphiquement avec les horloges qu'il supporte.

La classe CWall est déclarée dans le fichier Wall.h.
