Description détaillée

template<typename T>
class CMatrice< T >

Classe de gestion d'une matrice.

La gestion des matrices est réalisée de façon générale à l'aide d'un template. Le paramètre du template défini le type des données stockées dans la matrice.

Remarques
Cette classe n'est pas du tout optimisée.
