Description détaillée

Représente le labyrinthe.

Le labyrinthe est composé d'un tableau de CCase à deux dimensions. L'ensemble des parcours possible dans le labyrinthe est constitué par les connexions des cases. Les cases sont connectées grâce à leur pointeurs internes.
